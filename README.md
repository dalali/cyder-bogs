# cyder-bogs

A browser-based single-player clone of the classic 1994 DOS game **Cyberdogs**.

Top-down shooter with mission-based gameplay, weapon shop, enemy AI, and multiple weapon types — all running in your browser with no server required.

## Status

🚧 In development

## Play

Open `index.html` in your browser, or use the run script:

```bash
./run.sh start
```

## Development

```bash
./run.sh help
```
